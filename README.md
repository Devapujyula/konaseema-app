# 🥥 Konaseema Snacks E-commerce App

A full-stack e-commerce web application built for showcasing and selling traditional Konaseema snacks online.

---

## 🚀 Features

- 🛍️ Product listing with categories (Sweets, Hot, Pickles, Karampodi)
- 🔍 Search and filter functionality
- 🛒 Add to cart with quantity management
- 💰 Dynamic total price calculation
- 💳 Razorpay payment integration (Test mode)
- 📦 Checkout and order placement
- 🗃️ Orders stored in MySQL database
- 📱 Responsive UI using Bootstrap

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Bootstrap
- Axios

**Backend:**
- Node.js
- Express.js

**Database:**
- MySQL

**Payment:**
- Razorpay
